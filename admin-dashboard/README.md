# Admin Dashboard

React-based admin dashboard for IntelliGuard fraud detection platform.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Recharts (for data visualization)

## Features (Planned)

- Real-time fraud metrics dashboard
- Transaction monitoring
- Rule management UI
- Alert management
- User management
- Model performance metrics
